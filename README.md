# gameflow-example-plugin

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

Essential parts are the keywords of the package containing `gameflow-plugin` and the package `@simeonradivoev/gameflow-sdk` being a peer dependency.
For more info on the SDK check out the [SDK repo](https://github.com/simeonradivoev/gameflow-deck/tree/master/src/packages/gameflow-sdk)
